  <action>
Create `convex/navBadges.ts` with a single query `getNavBadges` that returns:
```typescript
{ checkInDue: boolean; unreadTicketCount: number }
```

Logic:
- `checkInDue`: Check if check-in is NOT locked. Reuse the same logic from `checkIns.ts getLockStatus` — find latest check-in, calculate next date using `getCheckInFrequencyDays()`. If no check-in exists OR current time >= next check-in date, return `true`.
- `unreadTicketCount`: Count tickets where `userId === currentUser` AND `status === "coach_responded"`. These are tickets where the coach replied but the client hasn't read yet.
- If user is not authenticated, return `{ checkInDue: false, unreadTicketCount: 0 }`.
- Import `getAuthUserId` from `./auth` and `getCheckInFrequencyDays` from `./helpers`.

Create `apps/client/src/hooks/useNavBadges.ts`:
```typescript
import { useConvexAuth, useQuery } from "convex/react";
import { api } from "@/convex/_generated/api";

export function useNavBadges() {
  const { isAuthenticated } = useConvexAuth();
  const badges = useQuery(api.navBadges.getNavBadges, isAuthenticated ? {} : "skip");
  return badges ?? { checkInDue: false, unreadTicketCount: 0 };
}
```
